# Colors

Design tokens defined in `src/styles/theme.css` and exposed as CSS variables. Toolbar's theme switcher swaps light/dark by toggling `class="dark-mode"` on `<html>`, so every swatch on this page reflects the current mode.

Use these via Tailwind utilities (`bg-primary`, `text-secondary`, `border-error`, etc.) — never hard-coded hex.

---

## Brand scale

Primary brand palette. Each step is a CSS var (`var(--color-brand-N)`) and the matching Tailwind utility is `bg-brand-N` / `text-brand-N`.
